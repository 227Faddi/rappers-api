Below is an explanation of each field:

  • id: The unique identifier for the rapper.
  • name: The full name of the rapper.
  • dateOfBirth: The date of birth of the rapper.
  • age: The current age of the rapper, calculated based on the date of birth.
  • nationality: The nationality of the rapper.

Usage

You can interact with the Rappers API using the following endpoints:

  • GET /api/rappers/:rapperName
    Retrieves data about a specific rapper.
    Example: https://rappers-api-l2wq.onrender.com/api/rappers/eminem
  • GET /api/rappers
    Retrieves data for all rappers.
    Example: https://rappers-api-l2wq.onrender.com/api/rappers
